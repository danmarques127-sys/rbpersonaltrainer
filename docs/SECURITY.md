# Security Policy

## Supported Versions
This is a static website repository. Security updates are applied to the production deployment as needed.

## Reporting a Vulnerability
If you discover a security issue (e.g., XSS vectors via embedded scripts, unsafe third-party embeds, or misconfigured headers), please report responsibly:

- Open a private communication with the maintainer via GitHub profile:
  https://github.com/danmarques127-sys

When reporting, include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment (what can be exploited)
- Any suggested fix (if available)

## Scope Notes
- This project is primarily static (HTML/CSS/JS).
- Production uses HTTPS (SSL) and may apply additional security headers via Apache/.htaccess depending on server compatibility.
